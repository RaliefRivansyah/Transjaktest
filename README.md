# Transjaktest
TransjakTest adalah aplikasi frontend berbasis React.js yang dirancang untuk menampilkan dan mengelola data armada kendaraan Transjakarta. Aplikasi ini terintegrasi dengan REST API untuk menerima dan menampilkan data kendaraan secara dinamis.

Proyek ini dibuat sebagai bagian dari tes teknis untuk posisi Frontend Engineer.

# Teknologi yang Digunakan
- React.js – Library utama untuk membangun antarmuka pengguna
- Functional Components & Hooks – Pendekatan modern dalam pengembangan React
- REST API – Untuk integrasi dan pengambilan data kendaraan
- Tailwind CSS – Utility-first CSS framework untuk styling

# Instalasi dan Menjalankan Proyek

Ikuti langkah berikut untuk menjalankan proyek secara lokal:

1. Clone Repository
git clone <url-repository>

2. Masuk ke Folder Project
`cd transjaktest`

3. Install Dependencies
`npm install`

4. Jalankan Development Server
`npm run dev`


Aplikasi akan berjalan di:

http://localhost:5173

(Port dapat berbeda tergantung konfigurasi Vite atau environment)
